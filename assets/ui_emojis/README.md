# UI emoji source files

Save the upcoming custom UI emoji artwork here using these exact names:

- `ticket_available.png`
- `ticket_used.png`
- `boss_appeared.png` — sword/new boss artwork
- `boss_escaped.png`
- `boss_defeated.png`

Use square transparent PNG files. Keep the important artwork centered with enough
transparent padding that it stays readable when Discord displays it at emoji size.

These files are source artwork only. The later UI update will upload them as Discord
application emojis and connect their resulting emoji IDs to the bot. Do not replace
the existing `assets/hp_digits/` templates.
